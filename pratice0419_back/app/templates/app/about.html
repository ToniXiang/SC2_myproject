關於我們

歡迎來到購物平台！這是一個整合了最新技術的全端專案，旨在提供使用者流暢的購物體驗。

專案背景

本平台使用 Flutter 作為前端框架，提供響應式設計，確保在不同裝置上皆能良好呈現。後端則採用 Django REST Framework，搭配 PostgreSQL 作為資料庫，確保資料的穩定性與安全性。

技術架構

  • 前端：Flutter，支援 Web、iOS、Android 平台。
  • 後端：Django REST Framework，提供 API 服務。
  • 資料庫：PostgreSQL，管理使用者與商品資料。
  • 部署平台：Render，簡化後端與前端的部署流程。

功能特色

  • 使用者註冊與登入功能。
  • 商品列表展示，包含名稱、圖片與價格。
  • 購物車功能，方便使用者管理商品。
  • 響應式設計，適配不同裝置。

介面特色

  • 支援深色與淺色模式，提升使用者體驗。
  • 固定視窗大小與對齊功能，提供更靈活的操作方式。
  • 簡潔直觀的使用者介面，易於上手。

開發環境

本專案使用以下技術棧進行開發：

  • Flutter
  • Django
  • PostgreSQL
  • Render 部署平台

挑戰與收穫

在開發過程中，我們克服了多項技術挑戰，包括插件配置、環境變數設定以及部署流程的優化。這些經驗讓我們對全端架構與雲端部署有了更深的理解。

感謝您的支持，期待您在平台上有愉快的購物體驗(不會實際購買，僅模擬)！
